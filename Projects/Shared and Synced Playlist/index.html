Shared and Synced Playlist Pipeline

Idea

A website that lets a group of people listen to the same songs at the same time. The playlist is created by users adding the next video's name in the comments section.

Steps

- Embed a Youtube video into a webpage using the Youtube API

- Add a button to allow you to 'sync' your listening to the master lisener

- Dynamically load video's depending on a variable text field value

- Find the video ID from video names

- Add a commenting section that allows visitors to add song suggestions

- Develop algorithm to identify most popular song to be added next
